# DALILA RESTO

### Paso 1 Crear una base de datos con un nombre, por ejemplo: resto

### Paso 2 crear un archivo .env con:

`
DB_USER=
DB_PASSWORD=
DB_HOST=
DB_PORT=
DB_NAME=resto
PORT=
`
### Paso 3 Ejecutar 

`node database/initializeDataBase.js`

Para crear las tablas y los datos de prueba
```
user@mail.com
user
1234

admin@mail.com
admin
acamica
```

### Paso 4 Ejecutar 

`npm i`

### Paso 5 Ejecutar 

`npm start`
